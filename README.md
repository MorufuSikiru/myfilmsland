# 🎬 Filmland Movie Website

Filmland is a modern responsive movie website built with HTML, CSS and JavaScript.  
It allows users to browse movies, search for titles, explore trending content, watch trailers and subscribe to newsletters.

---

# 🚀 Features

✅ Responsive Navigation Bar  
✅ Mobile Hamburger Menu  
✅ Smooth Scrolling Navigation  
✅ Movie Search Functionality  
✅ Trending Movies Section  
✅ Top Rated Movies Section  
✅ Trailers Section  
✅ Newsletter Subscription  
✅ Responsive Movie Grid Layout  
✅ Hover Effects & Modern UI  
✅ Netflix-style Design

---

# 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
